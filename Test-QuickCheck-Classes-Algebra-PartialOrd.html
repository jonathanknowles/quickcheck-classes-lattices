partialOrdLaws :: forall a. (Arbitrary a, Show a, Monoid a, PartialOrd a) => Proxy a -> Laws Source #

Laws for instances of PartialOrd.

Tests the following properties:

a `leq` a
a `leq` b && b `leq` a ==> a == b
a `leq` b && b `leq` c ==> a `leq` c
